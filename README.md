# 보행자 감지 스마트 LED 횡단보도

초음파 센서를 활용해 보행자 접근을 감지하고, LED로 시각적 경고를 제공하는 아두이노 기반 프로토타입입니다.

## 기술 스택
- Arduino IDE
- C/C++

## 주요 기능
- 초음파 센서를 이용한 거리 측정
- 보행자 접근 여부 판단
- 접근 시 LED 점등
- 작동 테스트 및 시연

## 담당 업무
- Arduino Uno에 LED와 초음파 센서를 연결하고 제어 코드 작성
- 초음파 센서 거리 데이터를 바탕으로 보행자 접근 여부 판단 로직 구현
- 센서 입력값에 따라 LED가 반응하도록 제어 기능 구현
- 하드웨어와 소프트웨어를 연동해 작동 테스트 및 시연 환경 구성

## 주요 성과
- 보행자 접근 시 LED가 반응하는 프로토타입 구현
- 하드웨어 조립, 센서 연동, 제어 로직 구현, 작동 테스트까지 전 과정을 직접 수행
- 우드보드 시연용 프로토타입을 제작해 실제 동작 가능한 형태로 완성
